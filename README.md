# Federated Learning & IoT Botnet Detection (Aishwarya S P)

**Project:** Federated Learning Framework for Resource-Constrained IoT Devices — implementation & dataset experiments.

## Overview
- Uses UNSW-2018 IoT Botnet dataset to detect DDoS attacks.
- Balanced subsample used for experiments (2000 rows: 1000 Normal / 1000 DDoS).
- Code includes preprocessing, NN training, simple FL prototype (Flower), and attack simulations.
